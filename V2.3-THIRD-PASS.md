# Periodic 2.3 — Third Pass

This pass keeps the v2.2 periodic-table arrangement and replaces the fake canvas-only backdrop with a real Filament-rendered GLB science laboratory.

## Major changes
- New room-scale GLB environment: floor, walls, ceiling, holographic platform, illuminated strips, molecule pedestals, display panels, portal frames, and ceiling rings.
- The periodic table overlay is transparent enough for the 3D room to remain visible.
- Slow cinematic camera drift creates true parallax behind the table.
- Stronger key and fill lighting reveal the environment instead of leaving it black.
- Existing v2.3 Explore laboratories, Journey Into Matter, isotope, ion, matter, nucleus, and compare features are retained.

## Honest implementation status
The background is now actual 3D geometry. It is not yet a fully walkable first-person laboratory, and the science words are still not extruded 3D lettering. Those require additional mesh assets or runtime text-to-mesh generation.
